Project 3: Using MIDI to control sound/music

For this assignment, we used Max along with a MIDI controller to control sounds. For this, I used a tri~ object as the base, and added some fun ways to control the sound. For example, you can control the amplitude of the wave by hitting the keys softer or lighter, and the pitch and modulation wheels do exactly what you'd think they do. It's pretty basic and not very interesting to talk about, but still, I'm proud of what I was able to accomplish with it. It probably gave me more of a headache than it should have, though!

Project 4: Algorithmic Composition

This assignment had us create a program that generates music in a rhythmic way. For this, I used Max to make a program that can play several chords in MIDI, and then plays more notes randomly based on those chords. Basically, there's a set of notes possible for each chord, and the note played is chosen randomly from that set. I made options so that you can make the chords play in a progression that I've set, or also make them random. Sometimes the melody the algorithm ends up Playing is pretty nice, but if I had more time I'd like to add more possible chords for the program to play around with.

Project Progress Report

So far, I've been able to set up a method where all the user has to do is click a button, and it switches what instrument is playing. Click the button below the instrument, and it'll switch to instrument number 0, 1, 2, 3, 4, or 5. Also, a unique animation plays for each instrument. With all of these, it's possible to set up a little performance! I set most of this up using Max logic and objects like ggates and gswitches, some if statements, and a ton of cords. I also used a metronome and fpic for the animations, and I used some simple math to get the animations to play in sync to the tempo. The next big thing to accomplish would be to set up different mixers for each instrument: things like panning, reverb, chorus, flanger, or other effects like that. Some of my instruments already have these effects applied, so I might have to re-record them completely dry. Also, I want to make it so that you can change the tempo of the playback, and the animations will slow down too. I think I can do that with some simple math. Finally, the only other thing I have to do is make an animation for when an instrument is off, and make sure that all the different animations are synced up perfectly. I think progress is going pretty well!
